Spotify IoT Player

Securely link your device using your own Spotify API credentials.

How to setup?

  1. Go to the Spotify Developer Dashboard.
  2. Click on "Create app".
  3. Enter an app name and description.
  4. Add this Redirect URI:
    https://spotify-setup-tool.vercel.app/api/callback
